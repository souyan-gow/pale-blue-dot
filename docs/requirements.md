作成日: 2025年

## 1. プロジェクト概要

### 1-1. ゲーム概要

| 項目 | 内容 |
|------|------|
| タイトル | PALE BLUE DOT |
| サブタイトル | A Journey Through the Solar System — and Beyond |
| ジャンル | 宇宙探索・アンビエント体験ゲーム |
| プラットフォーム | ブラウザ（HTML5 / Three.js） |
| 対象プレイヤー | 宇宙に感動できる人すべて |
| コアコンセプト | 勝ち負けなし。失敗なし。ただ、旅する。そして、帰る。 |

### 1-2. コンセプト

カール・セーガンの「Pale Blue Dot」——1990年にボイジャー1号が60億km彼方から撮影した地球の写真を起点に、宇宙の壮大さと地球のかけがえのなさを「体験」として届けるゲーム。

**旅の構造（円環）：**
```
地球を出発 → 太陽系を旅する → ヘリオポーズを越える
→ ケンタウルス座α星系へ到達 → 地球へ帰還 → 大気圏突入 → エンディング
```

### 1-3. 設計の優先順位

1. **科学的正確さ** — 実際のNASAデータ・AU比・体積比を使用。妥協しない
2. **スケールの体感** — 数字でなく「光が届く時間」「移動時間」で距離を表現
3. **美しさの演出** — プロシージャルテクスチャ・リアルタイム光計算
4. **孤独と静寂** — 木星以遠は音楽が消え、宇宙の静寂が体を包む

## 2. スケール仕様（科学的データ）

### 2-1. 惑星スケール

Earth radius = 1.0 scene unit（実際: 6,371 km）  
Sun radius = 109.3 scene units（実比率: 696,340 / 6,371 = 109.3×）  
1 AU = 409.3 scene units（地球軌道半径）

| 惑星 | 半径 (scene) | 軌道半径 (scene) | 実際のAU |
|------|-------------|-----------------|---------|
| 太陽 | 109.3 | 0 | — |
| 水星 | 0.383 | 158.4 | 0.387 |
| 金星 | 0.950 | 295.9 | 0.723 |
| 地球 | 1.000 | 409.3 | 1.000 |
| 火星 | 0.532 | 623.8 | 1.524 |
| 木星 | 10.973 | 2,129.6 | 5.203 |
| 土星 | 9.140 | 3,903.5 | 9.537 |
| 天王星 | 3.981 | 7,854.5 | 19.19 |
| 海王星 | 3.865 | 12,307.7 | 30.07 |
| ヘリオポーズ | — | 49,128 | 120 AU |
| ケンタウルス座α | — | 109,000,000 | 265,000 AU（4.24光年） |

### 2-2. スケール段階

| ステージ | 距離範囲 | 見えるもの |
|---------|---------|---------|
| 地表視点 | 〜数km | 惑星地形・空・他惑星が点 |
| 惑星軌道 | 〜数万km | 惑星全体・リング・衛星 |
| 内太陽系 | 〜5 AU | 内惑星の公転 |
| 外太陽系 | 5〜120 AU | 巨大惑星・太陽が星になる |
| 太陽圏外縁 | 120 AU〜1光年 | 地球がPale Blue Dotに |
| 近傍星間空間 | 1〜4.24光年 | 太陽が普通の星に |
| ケンタウルス座α | 4.24光年 | 太陽系がPale Yellow Dotに |

## 3. 機能要件

### 3-1. 航行システム

| 機能 | 要件 | 優先度 |
|------|------|--------|
| 目的地選択 | 全惑星＋ヘリオポーズ＋ケンタウルス座αを選択可能 | 必須 |
| カメラ移動 | 目的地へのスムーズなカメラアニメーション | 必須 |
| 軌道制御 | ドラッグ回転・ホイールズーム | 必須 |
| 時間加速 | ×0〜×100,000のリアルタイム制御 | 必須 |
| コックピット視点 | 着陸後・帰還中の一人称視点 | 必須 |
| ホーマン遷移 | 軌道力学的に正確な惑星間移動（視覚的表現） | 推奨 |
| 重力アシスト | 木星・土星通過時の加速演出 | 推奨 |

### 3-2. 惑星表現

| 惑星 | 必須要件 | 推奨要件 |
|------|---------|---------|
| 太陽 | 周縁減光・グロー | 黒点・フレア・グラニュレーション |
| 水星 | クレーター・灰色岩盤 | エジェクタ光条 |
| 金星 | 硫酸雲の渦 | 極渦 |
| 地球 | 大陸・海洋・極冠・雲レイヤー | バイオーム別色・大気圏リム |
| 火星 | 赤い荒野・極冠 | バレス・マリネリス・タルシス |
| 木星 | 縞模様・大赤斑 | Oval BA・多層ベルト |
| 土星 | 縞模様・多層リング（C/B/Cassini/A/F環） | 極六角形・パーティクルベルト |
| 天王星 | 薄青・極冠 | 縦向きリング |
| 海王星 | 深い青・縞模様 | 大暗斑・スクーター |

### 3-3. 特別シーン（必須）

| シーン | トリガー | 内容 |
|--------|---------|------|
| 太陽が星になる | 木星軌道通過後 | BGMフェード・テキスト表示 |
| Pale Blue Dot | 海王星軌道付近 | 地球が点に・セーガンの言葉 |
| ヘリオポーズ通過 | 120 AU到達 | 全音消滅・「恒星間空間」メッセージ |
| ケンタウルス座α到達 | 目的地到達 | 帰還/前進の選択 |
| 帰還モンタージュ | 帰還選択後 | 旅したすべての場所を定点カメラで |
| 大気圏突入 | 地球帰還時 | 炎エフェクト・振動・速度表示 |
| エンディング | 着地後 | セーガンの言葉フル版・写真スライドショー |

### 3-4. 写真システム

| 機能 | 要件 |
|------|------|
| 撮影 | いつでも現在の視点をキャプチャ |
| メタデータ | 撮影場所・太陽からの距離・AU・日時 |
| 自動タイトル | 状況から詩的なタイトルを生成 |
| アルバム | 撮影済み写真一覧・Pale Blue Dot図鑑（12枠） |
| エンディング連携 | 撮影した写真がエンディングスライドショーに使われる |

**Pale Blue Dot図鑑 12枠：**
1. 水星地表からの地球
2. 金星雲頂からの地球
3. 火星地表からの地球（キュリオシティが実際に撮影）
4. 木星軌道からの太陽系
5. 土星リングから（カッシーニが2013年に撮影）
6. 天王星軌道からの太陽系
7. 海王星軌道からのPale Blue Dot
8. ヘリオポーズからの太陽系
9. ケンタウルス座αからのPale Yellow Dot
10. 地球軌道からの月とのペア
11. 大気圏突入中
12. 地球帰還後・青空

### 3-5. サウンド（推奨）

| 場面 | サウンド設計 |
|------|------------|
| 地球付近 | 壮大なオーケストラ |
| 惑星地表（地球） | 唯一、生命の音（風・鳥・波） |
| 土星の嵐 | 嵐を鎮めると静寂→リング出現 |
| 木星以遠 | BGMフェードアウト→宇宙ドローン |
| ヘリオポーズ | 完全な沈黙（1秒）→静かなフィナーレ |
| 帰還モンタージュ | 旅のテーマが回帰 |
| 大気圏突入 | 轟音・炎・振動 |
| エンディング | 風の音・生命の音 |

## 4. 地表ビュー要件

各惑星に着陸すると地表ビューが表示される。

| 惑星 | 空の色 | 特徴的な演出 |
|------|--------|------------|
| 水星 | 漆黒（昼間も星空） | 太陽が地球の3倍大きく見える |
| 金星 | オレンジ〜茶 | 硫酸の霧・高圧大気の音 |
| 地球 | 青空 | 生命の音・雲・緑の地形 |
| 火星 | ピンク〜赤茶 | 砂嵐パーティクル・赤い岩 |
| 木星 | 黄金〜茶 | 嵐の雲が渦巻く |
| 土星 | 黄土〜茶（嵐中） | 「嵐を鎮める」ボタン→空にリング出現 |
| 天王星 | 薄青 | 極めて静か |
| 海王星 | 深い青 | 最強の嵐・太陽が星の1つ |
| プロキシマb | 赤茶（仮説） | 赤色矮星の赤い空（科学的仮説として明記） |

## 5. 非機能要件

| 項目 | 要件 |
|------|------|
| パフォーマンス | PCブラウザで60fps維持（Chrome/Firefox/Safari最新版） |

| 外部依存 | Three.js r128（CDN）のみ |

## 6. デプロイ要件

### 6-1. 最小構成（静的ホスティング）

```
index.html          ← メインゲームファイル（単一HTMLで完結）
```

Three.jsはCDN（cdnjs.cloudflare.com）から読み込むため、
追加ファイルなしで動作する。

### 6-2. 推奨ホスティング

- **Vercel** — `vercel deploy`一コマンドで完了
- **Netlify** — ドラッグ&ドロップでデプロイ可能
- **GitHub Pages** — リポジトリにpushするだけ

### 6-3. 将来的なサーバー機能（任意）

写真のクラウド保存・リーダーボード等が必要な場合は以下を検討：
- Supabase（PostgreSQL + Storage）
- Firebase
